# 🌿 AI Crop Disease Detection

AI-powered crop disease detection web application using TensorFlow, Flask, and MobileNet for plant leaf image classification.

## 🔗 Live Demo

https://ai-crop-disease-detection-eohw.onrender.com

## 📌 Project Overview

This project is an AI-powered web application that detects crop diseases from plant leaf images using a MobileNet-based Convolutional Neural Network (CNN). Users can upload an image through the web interface, and the system processes it to identify the disease with a confidence score. The project demonstrates the integration of deep learning, image preprocessing, backend development, and cloud deployment.

## ✨ Features

- 🌿 Detects crop diseases from plant leaf images
- 🤖 MobileNet-based deep learning model
- 🖥️ Flask web application
- 📷 Image upload interface
- ⚡ Image preprocessing and prediction pipeline
- ☁️ Deployed on Render
- 📂 Version controlled using Git and GitHub
