# AI Crop Disease Detection
AI-based web app to detect crop diseases using deep learning.

# Live Demo
Live App: https://your-render-url.onrender.com

# Deployment Note
Due to platform limitations, the TensorFlow model is not deployed online.
The full model runs locally and achieves high accuracy.

# Tech Stacks
Frontend: HTML, CSS
Backend: Flask
ML: TensorFlow, CNN (MobileNet)
Deployment: Render

# Project Description
This project presents a production-oriented AI system for automated crop disease detection using deep learning. A MobileNet-based convolutional neural network was trained on a multi-class plant disease dataset to classify various crop conditions with high accuracy. The backend was developed using Flask, exposing RESTful endpoints for image-based predictions, while the frontend enables seamless user interaction through image uploads. The system includes efficient preprocessing pipelines, modular architecture, and clean separation of concerns. The application is deployed on Render for accessibility, with model inference handled locally due to cloud environment limitations. The project demonstrates practical integration of machine learning, backend development, and deployment workflows.
